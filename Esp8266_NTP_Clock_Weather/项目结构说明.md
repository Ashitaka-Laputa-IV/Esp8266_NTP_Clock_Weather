# ESP8266 NTP时钟天气站 - 项目结构说明

## 项目概述

这是一个基于ESP8266的智能时钟天气显示系统，具有以下功能：
- 显示当前时间和日期（通过NTP同步）
- 显示当前天气信息（通过和风天气API）
- 显示未来3天天气预报
- 支持WiFi自动配置和网页配网
- 使用DS18B20温度传感器监测环境温度
- OLED显示屏输出

## 文件结构

```
Esp8266_NTP_Clock_Weather/
├── Esp8266_NTP_Clock_Weather.ino    # 重构后的主程序（模块化版本）
├── main_simple.ino                   # 简化版主程序（易于阅读）
├── config.h                          # 配置文件（参数和常量）
├── display_manager.h                 # 显示管理器头文件
├── display_manager.cpp               # 显示管理器实现
├── HeFeng.h                          # 和风天气API客户端头文件
├── HeFeng.cpp                        # 和风天气API客户端实现
├── WeatherStationFonts.h             # 字体定义（外部库）
├── WeatherStationImages.h            # 图像定义（外部库）
└── 项目结构说明.md                   # 本文档
```

## 各文件功能说明

### 1. 主程序文件

**Esp8266_NTP_Clock_Weather.ino** - 重构后的模块化版本
- 采用模块化设计，代码结构清晰
- 遵循Google C++代码规范
- 包含详细的注释说明
- 适合项目维护和扩展

**main_simple.ino** - 简化版本
- 代码结构简洁，易于理解
- 所有功能集成在一个文件中
- 适合初学者学习和快速上手
- 注释详细，逻辑清晰

### 2. 配置文件

**config.h** - 系统配置参数
- WiFi配置：SSID、密码
- 天气API配置：密钥、城市代码
- 硬件配置：引脚定义、显示地址
- 时区配置：时区设置、夏令时
- 更新间隔：天气、温度更新频率

### 3. 显示管理模块

**display_manager.h/cpp** - OLED显示管理
- 显示初始化和管理
- 各种显示页面的绘制
- 进度条显示
- UI状态管理
- 支持多帧切换动画

### 4. 天气API模块

**HeFeng.h/cpp** - 和风天气API客户端
- HTTPS请求处理
- JSON数据解析
- 天气数据提取和格式化
- 错误处理和重试机制
- 天气图标映射

### 5. 资源文件

**WeatherStationFonts.h/WeatherStationImages.h** - 字体和图像资源
- OLED显示所需的字体定义
- 天气图标和符号
- 由外部库提供，无需修改

## 代码规范特点

### 命名规范
- 变量：`snake_case`（如：`current_temperature`）
- 常量：`kCamelCase`（如：`kUpdateInterval`）
- 函数：`PascalCase`（如：`UpdateWeatherData`）
- 类：`PascalCase`（如：`DisplayManager`）

### 代码格式
- 缩进：2个空格
- 大括号：K&R风格
- 注释：中英文双语，详细说明
- 头文件顺序：系统头文件 → 库头文件 → 项目头文件

### 模块化设计
- 功能分离：显示、网络、数据各司其职
- 接口清晰：模块间通过明确定义的接口通信
- 易于测试：各模块可以独立测试
- 便于维护：修改一个模块不影响其他部分

## 硬件连接

```
ESP8266引脚配置：
- D2 (GPIO4)  -> OLED SDA
- D5 (GPIO14) -> OLED SCL
- D7 (GPIO13) -> DS18B20数据线
```

## 使用说明

### 1. 配置参数
在`config.h`文件中设置：
- WiFi名称和密码
- 和风天气API密钥
- 城市代码（如："beijing"）
- 时区设置

### 2. 编译上传
- 使用Arduino IDE或PlatformIO
- 选择正确的开发板（NodeMCU或Wemos D1 Mini）
- 安装必要的库：
  - ArduinoJson
  - ESP8266WiFi
  - SH1106Wire/SSD1306Wire
  - OLEDDisplayUi

### 3. 运行流程
1. 设备启动后尝试自动连接WiFi
2. 如果自动连接失败，进入网页配网模式
3. 连接NTP服务器同步时间
4. 获取天气数据并显示
5. 循环更新显示内容

## 故障排除

### 常见问题
1. **WiFi连接失败**：检查SSID/密码，信号强度
2. **天气数据获取失败**：检查API密钥和网络连接
3. **显示异常**：检查I2C连接和引脚定义
4. **时间同步失败**：检查网络连接和NTP服务器

### 调试技巧
- 启用串口调试输出
- 检查HTTP响应状态码
- 验证JSON数据格式
- 测试硬件连接
